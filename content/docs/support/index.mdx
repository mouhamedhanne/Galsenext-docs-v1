# Support and Contact

## 1. Getting Help

To get help, first check the [official Galsenext documentation](https://galsenext-docs.vercel.app/docs). If you cannot find an answer to your question:

- **GitHub Issues**: Open an issue on [our GitHub page](https://github.com/mouhamedhanne/Galsenext) by describing your problem or question.
- **GitHub Discussions**: Participate in discussions to exchange ideas and ask questions.
