# Installation

Pour commencer avec **Galsenext**, veuillez vous assurer que votre environnement de développement respecte les prérequis suivants :

## Prérequis

**Node.js** : Version **20** ou plus récente. Pour vérifier votre version actuelle de Node.js, exécutez la commande suivante dans votre terminal :

```bash
node -v
```

## Base en Next.js

Il est recommandé d'avoir une bonne compréhension de Next.js pour tirer pleinement parti de ce package. Familiarisez-vous avec les concepts de base tels que les pages, les composants, et le routage.

Installer **Galsenext**:

```bash
npm install galsenext
# ou
yarn add galsenext
```
