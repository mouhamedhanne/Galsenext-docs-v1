# Bienvenue sur Galsenext

Galsenext est un `starter kit` puissant basé sur [Next.js](https://nextjs.org) et [Neon](https://neon.tech) qui offre une
solution prête à l'emploi pour le développement d'applications web
modernes. Conçu pour accélérer le processus de création d'applications,
**Galsenext** inclut une configuration complète avec **NextAuth** pour
l'authentification, **Prisma** et **PostgreSQL** pour la gestion des données,
ainsi qu'une internationalisation prête à l'emploi (`i18n`).

## Fonctionnalités Principales

- **Authentification et Autorisation** : Gestion de l'authentification utilisateur avec NextAuth, incluant les stratégies OAuth, Email, et Credentials.
- **Prisma ORM** : Intégration de Prisma pour la manipulation efficace des données avec PostgreSQL.
- **Support Neon** : Intégration avec Neon pour une base de données PostgreSQL performante et évolutive.
- **Internationalisation (i18n)** : Prise en charge multilingue pour offrir une expérience utilisateur accessible à tous.
- **Onboarding Utilisateur** : Système d'intégration guidée pour simplifier l'expérience utilisateur dès le premier jour.
- **Support TypeScript** : Utilisation de TypeScript pour une expérience de développement plus robuste et sécurisée.

## Pourquoi Utiliser Galsenext ?

- **Gain de Temps** : Une configuration initiale clé en main qui vous permet de vous concentrer sur les fonctionnalités spécifiques de votre projet.
- **Flexibilité** : Adaptez et étendez le starter kit pour répondre aux besoins uniques de votre application.
- **Code de Qualité** : Bénéficiez des meilleures pratiques en matière de développement web moderne, avec des outils comme React, Next.js, et Prisma.

## Commencer avec Galsenext

1.  **Installation Rapide** : Utilisez la commande suivante pour installer Galsenext et démarrer votre projet :

```bash
npm install galsenext
# ou
yarn add galsenext
```

2.  **Personnalisation** : Explorez les fichiers de configuration pour adapter le package à vos besoins spécifiques.
3.  Déploiement Facile : Déployez votre application sur n'importe quelle plateforme compatible avec Next.js, comme Vercel
